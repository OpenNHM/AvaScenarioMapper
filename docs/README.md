### Overview

The **Avalanche Scenario Mapper Preview** is an interactive, map-based visualization tool designed to explore and evaluate avalanche scenarios generated by the Avalanche Scenario Model Chain and Mapper pipeline.

### Key Features
- **Interactive Web Map**  
  View winter and spring avalanche scenarios directly in the browser.

- **Release / Runout Filtering**  
  Toggle between release areas (REL), runout zones (RES), and volume-based size classes.

- **Transparency Control**  
  Fine-tune map visibility with adjustable opacity presets.

- **Scenario Switching**  
  Instantly switch between *Winter (New Snow)* and *Spring (Wet Snow)* scenario.

- **Hillshade & Basemaps**  
  Choose between Carto Light, Bergfex Topo, or combined hillshade imagery.

- **Metadata Popups**  
  Click any avalanche outline to reveal model attributes (e.g., sector, flow type, size class, potential).
